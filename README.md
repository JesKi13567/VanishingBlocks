# Vanishing Blocks -- 消失的方块
## 当前版本：v1.0 for MC1.20.0-4
### 简介
世界上的方块突然开始按种类慢慢消失，

而打破这一诅咒的方式是击败末影龙！

玩家们能否团结一心，成功通关呢？

### 说明
特别地，禁用消失的方块有：

黑曜石、下界传送门、末地传送门框架、末地传送门。

你可以在菜单中设置：
- 禁用消失方块：基岩、刷怪笼、白色床、工作方块（4 种，工作台、熔炉、锻造台、切石机）。
- 死亡不掉落、永久夜视、方块补给。
- 消失间隔、显示计数（、预设）。

注：预设为 MC1.20.0-1 版本设计，而 MC1.20.2+ 将使用数据包产生随机。

最后，你还可以使用包内自带的py文件生成你自己的预设！

玩的开心！
